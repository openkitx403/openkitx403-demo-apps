Quick Start

1

Generate Keypair

Create a new Solana keypair for bot authentication.

solana-keygen new --outfile bot/keypair.json
2

Install Dependencies

Install required Python packages for the bot.

cd bot && pip install -r requirements.txt
3

Run the Bot

Start the autonomous trading bot.

python bot.py
